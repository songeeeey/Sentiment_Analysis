# 네이버 영화 리뷰 감성 분석 📽️

### 1. 사용한 데이터 셋 📁

제가 사용한 데이터 셋은 네이버 영화 리뷰 데이터 셋입니다. 

|항목|설명|
|---|---|
|출처|[Naver Sentiment Movie Corpus (NSMC)](https://raw.githubusercontent.com/e9t/nsmc/master/ratings_train.txt)|
|크기|약 78MB|
|train data|150,000개|
|test data|50,000개|
|언어|한국어|
|주요 컬럼|`id`, `document`, `label`|
